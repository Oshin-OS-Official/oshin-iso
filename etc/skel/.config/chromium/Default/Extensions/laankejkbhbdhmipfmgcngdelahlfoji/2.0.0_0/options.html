  • Maximum Time Allowed Per Day

    Set the maximum number of minutes per day that you will be allowed to browse sites on the Blocked Sites list.

    minute(s)

    This time applies to all Blocked Sites in aggregate -- it is not a per-site timer. You cannot change this number once your time for the day has expired.

  • Active Days

    Set the days during which StayFocusd will restrict your browsing activity.

    You cannot turn the extension off for the current day.

  • Active Hours

    Set the hours during which StayFocusd will restrict your browsing activity. These hours will apply to all Active Days.

    If you change your Active Hours, the new hours will not take effect until this time tomorrow (24 hours from now)

  • Daily Reset Time

    Set the time at which StayFocusd will reset your allowed time for the day.

    Make sure to set your Daily Reset Time to a time after the end of your Active Hours.

    If you change your Daily Reset Time, the new time will not take effect until this time tomorrow (24 hours from now)

  • The Nuclear Option

    Activating this option will block sites for the number of hours you indicate, on the days you indicate, independent of your Active Hours. There is no way to cancel this once you activate it.

  • Customize

    Sync

    By default, StayFocusd will sync your Blocked Sites and Allowed Sites between all your devices via Chrome Sync. Click the box below to turn this feature off.

    Popup

    Infobar

    The infobar slides down from the top of a webpage to display important messages.

    StayFocusd update notices

    When StayFocusd updates, a NEW badge is displayed on the StayFocusd icon. Clicking the icon pops open a new window with info about the latest update. This happens very infrequently.

    Notifications

    Set up to five times you want to be warned before time runs out or before the Nuclear Option kicks in (both use the same settings). To disable notifications, leave all fields blank.

    Redirect message

    When StayFocusd blocks a site, it redirects to stayfocusd.com with the message "Shouldn't you be working?" You can replace that message with a custom message of your own.

    Data privacy

  • Export Settings

    You can export your settings as a backup, or to transfer them to another computer. You'll be prompted to save a .json file. Just click Save, and you're done.

    Import Settings

    If you previously exported your settings, click the Choose File button, locate the .json file, then click Import Settings.

    Importing settings ...

    IMPORTANT: Importing settings will overwrite any existing settings. There is no way to undo this.
